Sobre o AVDC

Princípios:

1. Os dados pertencem ao usuário.

2. O AVDC nunca altera os dados originais.

3. O AVDC possui apenas permissão de leitura sobre a fonte de dados.

4. Índices, cache e metadados ficam em diretório separado.

5. A IA é opcional e nunca requisito para o funcionamento.

6. Novas funcionalidades devem nascer como plugins.

7. O Core deve permanecer pequeno, estável e previsível.
